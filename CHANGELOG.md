## [0.1.0] - 2026-06-12

Initial release. System photo/video library picker for Mob apps.

- `MobPhotos.pick/2` opens the OS photo picker and returns the chosen media.
- Extracted from mob core in the 0.7.0 plugin-extraction wave.
- Requires `mob ~> 0.7`.
